9/13/10 Good-bye Program The Good-bye Program is intended to output good-bye in six different languages: English, Japanese, French, Spanish, Korean, or Chinese. When the appropriate button is pressed, the phrase is shown on the label above, and the country's flag is displayed at the center of the screen.
9/19/10 Mailing Label Program The Mailing Label program gets information typed in by the user and outputs it in a mailing label format. The program obtains the text in the text box and concatenates them together in the form of a mailing label. The Display Label button shows the text, and the Clear Text button wipes the text in the label for another set of information.
9/19/10 Help Page The Help Page program was made to replicate an official help page of a company. I made my own company, logo, and company background that is shown by the display logo button. The color and font buttons allow the user to customize the text they input in the text box and output using the submit button. The picture on the right hand side changes as the user clicks it.
9/30/10 Weather Report The Weather Report introduces the function of radio buttons. The user types in his/her name in the text box above first. By clicking one of the radio buttons, the program outputs the sentence "It looks like (weather) today, (name) substituting (weather) for the type of weather the radio button is checked and (name) for the user's name.
10/4/10 Car Rental The Car Rental program is a program that allows people who use it to rent cars and make payments by computer. The program requires the user to input their personal information like name and address. The user then selects the car they used and the extras they had. Below that the user inputs the beginning and ending odometer reading and the number of days they used the car. The calculate button then executes the calculations and outputs them in the labels below.
10/6/10 Body Mass Index Body mass index is the measure of someone's weight in relation to height. Therefore the BMI program calculates the user's body mass index according to the inputted information. The user types in his/her's height and weight to calculate the BMI. The equation is: weight in kilograms divided by height in meters squared. The program also keeps track of how many people calculated their BMI and the average of the total people's BMI.
10/18/10 Currency Exchange Program The Currency Exchange Program calculates whatever amount in American dollars to three other choices of money (based on the current time's exchange rates). The user types in the amount of money in dollars, and the amount will appear at the bottom with the currency's picture.
10/26/10 (12/7/10 revised) Test Score Program The Test Score Program gets two test scores and outputs the letter grade for each of the scores. The program shows the user the highest score out of the two and also the average of the two scores.
10/21/10 and 10/29/10 Craps Game (Dice Roll) The Craps Game program is a replica of the real Craps game played in casinos. The object of the game is fairly simple after you learn it once. If the first roll of the die luckily rolls to a total of 7, then the player immediately wins. If the total is a 2, 3, or 12 on the first roll, then the player loses. If it isn't any of the previous numbers, the program will put a 'point' on that number. A point in this game is like a marker. Continuing the rolls after getting a point on a number, if the die rolls a sum of the number with the marker, the player wins. If the die rolls to a 7 after the first roll, then the player loses.
11/8/10 T-Shirt Sales The T-Shirt Sales program simulates an online order taking program that a company who profits from selling items uses. Comsumers of the store's goods input their name, order number, and quantity of the clothing they want to buy. After selecting the size and extras, the customer will click Add to Current Order. This will add on to the current customer's bill until they have clicked Complete Order. A message box will appear with the total cost. The summary is for the manager of the store to check up on how much profit and clothing they have sold.
11/29/10 Slot Machine The Slot Machine program simulates a real slot machine. The jackpot has a starting amount of 1000 dollars in it. The user inputs the amount of money equal to the amount of credits he wants in the machine. The player then clicks the amount of dollars bet each time the lever is pulled. When the three pictures match as the lever is pulled, the player win the jackpot which is added to their credits. (soon to be revised)
12/7/10 Rose Bowl The Rose Bowl program shows the two schools competing against each other in the most recent Rose Bowl football game. This program is for an informational purpose of the history, schedule, and past scores. The history pages can be reached by clicking on the menu bar on the top of the program. It will drop down a list of the two schools and clicking either one will take you to the selected school's history page. The button Who Will Win? is my opinion on what team will win based on the information I have read.
